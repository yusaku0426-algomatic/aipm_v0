# 全社生成AI推進プロジェクト Phase2 ドキュメント管理

## プロジェクト概要
- **プロジェクト名**: 鴻池運輸株式会社 全社生成AI推進プロジェクト Phase2
- **期間**: 2025年7月1日 〜 2025年9月30日
- **総予算**: 22,935,000円

## PMBOKプロセス群による文書管理

### 1_initiating (立ち上げプロセス群)
- プロジェクト憲章
- ステークホルダー登録簿

### 2_discovery (発見・調査)
- 要件定義書
- 現状分析レポート
- 技術調査報告書

### 3_planning (計画プロセス群)
- プロジェクト管理計画書
- WBS
- スケジュール
- リスク管理計画書

### 4_executing (実行プロセス群)
- 開発成果物
- テスト結果
- 変更要求書

### 5_monitoring (監視・コントロールプロセス群)
- 進捗レポート
- 品質監査結果
- リスク管理ログ

### 6_closing (終了プロセス群)
- プロジェクト完了報告書
- 教訓集
- 成果物引き渡し書

## 文書管理ルール
1. 各フェーズの文書は対応するディレクトリに配置
2. 文書のバージョン管理はGitで実施
3. 主要な変更はPull Requestで管理
4. レビューは必須（最低1名の承認）
